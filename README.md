<p>
  O Pesquisa de Leis é uma aplicação web desenvolvida para agilizar o acesso à legislação brasileira. Focada na experiência do usuário, a plataforma centraliza as principais leis do país, permitindo buscas instantâneas e filtragem intuitiva por áreas do direito. Um dos grandes diferenciais é a integração com a API do Google Gemini, que atua explicando o conteúdo das leis de forma simples. Tecnologias utilizadas: HTML, CSS, JavaScript, JSON, Git, GitHub e Google Gemini
